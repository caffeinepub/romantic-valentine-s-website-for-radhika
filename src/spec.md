# Romantic Valentine's Website for Radhika

## Overview
A romantic interactive website with a "Will you be my Valentine?" theme centered around Radhika. The application features elegant animations, soft visual effects, and interactive elements that create a dreamy, love-themed atmosphere with a playful evasion mechanic, all contained within a centered box with full vertical scrolling functionality.

## Core Features

### Layout Structure
- All content contained within a centered box container for clean, balanced composition
- Full vertical scrolling functionality throughout the entire website on both mobile and desktop browsers
- Smooth vertical scrolling behavior up and down the page
- Clean, organized layout that maintains visual hierarchy within the centered container
- Content properly spaced and organized for optimal scrolling experience across all screen sizes

### Visual Design
- Romantic red gradient background using the romantic gradient background asset
- Clean interface without footer credits for a minimalist appearance
- Elegant typography with glowing text effects
- Large, prominent "Radhika" name text displayed with increased size to stand out more prominently with soft glowing effect
- "Radhika" text perfectly centered between two pink heart images in a single horizontal line alignment
- Multi-layered floating hearts animation throughout the interface using the floating heart asset
- Hearts should move smoothly across the interface within the centered container
- Hearts should vary in opacity and size for a romantic layered effect without obstructing main text
- Fade-in effects for content reveal
- Fully responsive design that works seamlessly on mobile devices with optimized text sizes, margins, and button layouts for smaller screens
- Enhanced mobile responsiveness with improved spacing, scaling, and animations for a polished, finished aesthetic
- Clean design optimized specifically for mobile view with proper touch targets and visual hierarchy
- Tidy, polished look consistent with romantic theme throughout all components
